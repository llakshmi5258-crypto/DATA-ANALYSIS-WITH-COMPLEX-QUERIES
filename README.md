##The task focuses on performing advanced data analysis using a relational database of customers and orders. It involves organizing the data into tables and inserting sample records to simulate real-world business transactions. The analysis leverages window functions to calculate rankings, running totals, and growth trends over time. CTEs (Common Table Expressions) are used to simplify complex queries and structure intermediate calculations. Subqueries help filter data and compute comparative metrics such as above-average spending. Customers are segmented by behavior, such as loyal, regular, or occasional, and top-performing months are identified. The goal is to uncover patterns, trends, and high-value customers. Finally, key performance indicators like total revenue, total orders, and average order value are summarized to provide actionable business insights.
